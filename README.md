## Install

1. Add this repository to your `composer.json`
```json
"repositories": [
    {
        "type": "git",
        "url": "https://github.com/figuren-theater/ft-logos"
    }
]
```

2. Install via command line
```sh
composer require figuren-theater/ft-logos
```
